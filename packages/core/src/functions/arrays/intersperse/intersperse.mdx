# intersperse

Insert new values between all existing values in an array.

## Parameters

| Parameter     | Type       | Required |
| :------------ | :--------- | :------: |
| **array**     | `any[]`    |    ✔     |
| **insertion** | `...any[]` |    ✔     |

## Example

With a single insertion:

<Example array="a, b, c, d" insertion="+" />

With multiple insertions:

<Example array="a, b, c, d" insertion="+, -" />

With a function:

<Source dark code={`
import { intersperse } from "@baggie/core";

const numbers = [10, 20, 30, 40];
const insertion = ({ previous, next }) => previous + (next - previous) / 2;

const newNumbers = intersperse(numbers, insertion);
// newNumbers = [10, 15, 20, 25, 30, 35, 40]
`} />
