# uppercaseFirstLetter

Convert the first letter in a string to uppercase.

## Parameters

| Parameter   | Type     | Required | Description                                                  |
| :---------- | :------- | :------: | :----------------------------------------------------------- |
| **string**  | `string` |    âœ”     | The string to manipulate.                                    |
| **options** | `object` |          | The default locale is the host environment's current locale. |

| Option            | Type               | Default value | Description                                      |
| :---------------- | :----------------- | :------------ | :----------------------------------------------- |
| **lowercaseRest** | `boolean`          | `false`       | Convert the rest of the characters to lowercase. |
| **locale**        | `string \| string[]` |               | Where the substring starts.                      |
