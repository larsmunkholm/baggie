# stripTagsFromString

Strip HTML and XML tags from a string. Tags are replaced with a space by default.

## Parameters

| Parameter       | Type      | Required | Description                                              |
| :-------------- | :-------- | :------: | :------------------------------------------------------- |
| **string**      | `string`  |    âœ”     | The string to strip for tags                             |
| **replaceWith** | `string`  |          | What to replace the tags with. Defaults to a space `" "` |
| **trim**        | `boolean` |          | Defaults to `true`                                       |
