# getPercentageBetween

Get the position between two numbers as a percentage.

## Parameters

| Parameter               | Type                           | Required | Default value |
| :---------------------- | :----------------------------- | :------: | :------------ |
| **position**            | `number`                       |    ✔     |               |
| **interval**            | `{ from: number; to: number }` |    ✔     |               |
| **percentageAsDecimal** | `boolean`                      |          | `true`        |
