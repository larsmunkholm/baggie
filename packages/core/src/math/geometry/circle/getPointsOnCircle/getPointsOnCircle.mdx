# getPointsOnCircle

Get an array of equally distanced points on the perimeter of a circle.

## Parameters

| Parameter          | Type     | Required |
| :----------------- | :------- | :------: |
| **radius**         | `number` |    ✔     |
| **numberOfPoints** | `number` |    ✔     |
| **offset**         | `number` |          |
