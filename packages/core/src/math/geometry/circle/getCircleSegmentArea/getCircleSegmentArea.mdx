# getCircleSegmentArea

Get the surface area of a segment of a circle given the radius
of the circle and the height of the segment.

## Parameters

| Parameter  | Type     | Required |
| :--------- | :------- | :------: |
| **radius** | `number` |    ✔     |
| **height** | `number` |    ✔     |
