# nonNumberFallback

Return 0 if input is not and can't be converted into a number.

## Parameters

| Parameter    | Type     | Required |
| :----------- | :------- | :------: |
| **value**    | `any`    |    ✔    |
| **fallback** | `number` |    ✔    |
