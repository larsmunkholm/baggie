# convertRangePosition

Convert the position in a range to another range.

## Parameters

| Parameter     | Type                           | Required |
| :------------ | :----------------------------- | :------: |
| **position**  | `number`                       |    ✔     |
| **fromRange** | `{ from: number; to: number }` |    ✔     |
| **toRange**   | `{ from: number; to: number }` |    ✔     |
