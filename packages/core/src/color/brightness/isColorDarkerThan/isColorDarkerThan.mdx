# isColorDarkerThan

Compare the darkness of two colors.

## Parameters

| Parameter  | Type     | Required | Description                 |
| :--------- | :------- | :------: | --------------------------- |
| **colorA** | `string` |    ✔     | The color to check.         |
| **colorB** | `string` |    ✔     | The color to check against. |
