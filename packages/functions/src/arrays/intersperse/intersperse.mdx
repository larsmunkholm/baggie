# intersperse

Insert new values between all values in an array.

## Parameters

| Parameter     | Type       | Required |
| :------------ | :--------- | :------: |
| **array**     | `any[]`    |    ✔     |
| **insertion** | `...any[]` |    ✔     |

## Example

With a single insertion:

<Example array="a, b, c, d" insertion="+" />

With multiple insertions:
