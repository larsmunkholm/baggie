# getPasswordRegex

Returns a regular expression to validate passwords with.

## Parameters

| Parameter   | Type     | Required |
| :---------- | :------- | :------: |
| **options** | `object` |          |

| Option                  | Type      | Default value                              |
| :---------------------- | :-------- | :----------------------------------------- |
| **minLength**           | `number`  | `8`                                        |
| **maxLength**           | `number`  |                                            |
| **minDigits**           | `number`  | `1`                                        |
| **minLowercase**        | `number`  | `1`                                        |
| **minUppercase**        | `number`  | `1`                                        |
| **minSpecial**          | `number`  | `1`                                        |
| **specialCharacters**   | `string`  | ``"!#$€%&§'()*+,./:;<=>?@[]\^_`´{\|}~¨-"`` |
| **forbiddenCharacters** | `string`  |                                            |
| **allowUnicode**        | `boolean` | `false`                                    |
