# getTagsRegex

Returns a regular expression for matching all HTML and XML tags in a string.

## Parameters

| Parameter               | Type      | Default value |
| :---------------------- | :-------- | :------------ |
| **groupSequentialTags** | `boolean` | `false`       |

## Example

<Example string="<p><b>Bold move.</b><br/>Paragraph.</p><!-- comment --> :)" />
