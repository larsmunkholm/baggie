# isPasswordValid

Check if a password is valid.

Accepts a string or an input element.

## Parameters

| Parameter    | Type                         | Required |
| :----------- | :--------------------------- | :------: |
| **password** | `string \| HTMLInputElement` |    ✔     |
| **options**  | `object`                     |          |

| Option                  | Type      | Default value                           |
| :---------------------- | :-------- | :-------------------------------------- |
| **minLength**           | `number`  | `8`                                     |
| **maxLength**           | `number`  |                                         |
| **minDigits**           | `number`  | `1`                                     |
| **maxDigits**           | `number`  |                                         |
| **minLowercase**        | `number`  | `1`                                     |
| **maxLowercase**        | `number`  |                                         |
| **minUppercase**        | `number`  | `1`                                     |
| **maxUppercase**        | `number`  |                                         |
| **minSpecial**          | `number`  | `1`                                     |
| **maxSpecial**          | `number`  |                                         |
| **specialCharacters**   | `string`  | ``"!#$%&'()*+,./:;<=>?@[]\^_`{\|}~¨-"`` |
| **forbiddenCharacters** | `string`  | `" "`                                   |
| **allowUnicode**        | `boolean` | `false`                                 |
