# splitAtSpace

Split a string at the nearest previous whitespace to a given a index.

Words will not get split.

## Parameters

| Parameter        | Type     | Required | Description                                             |
| :--------------- | :------- | :------: | :------------------------------------------------------ |
| **string**       | `string` |    ✔     | The string to split.                                    |
| **nearestIndex** | `number` |    ✔     | Split at the nearest previous whitespace to this index. |
