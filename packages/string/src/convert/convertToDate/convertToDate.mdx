# convertToDate

Convert a string (or an object) to a date.

Returns `undefined` if given an invalid date.

## Parameters

| Parameter   | Type                                                     | Required |
| :---------- | :------------------------------------------------------- | :------: |
| **date**    | `string \| { day: number; month: number; year: number }` |    âœ”     |
| **options** | `object`                                                 |          |

| Option             | Type      | Default value |
| :----------------- | :-------- | :------------ |
| **zeroBasedMonth** | `boolean` | `false`       |
| **monthBeforeDay** | `boolean` | `false`       |
