# convertEmailsToLinks

Find email addresses in a string and convert them to links.

## Parameters

| Parameter   | Type     | Required |
| :---------- | :------- | :------: |
| **string**  | `string` |    âœ”     |
| **options** | `object` |          |

| Option       | Type              | Descriptions                                                                                               |
| :----------- | :---------------- | :--------------------------------------------------------------------------------------------------------- |
| **subject**  | `string\|boolean` | Set to `true` to set the subject to `{document.title} @ {window.location.href}` or define a custom string. |
| **replacer** | `(args) => void`  | Define your own replace function.<br />`args` is an object matching `{ email: string; href: string }`      |
