# getHalfLifeDecay

Calculate how much a substance has decayed given it's half life and the time passed.

## Parameters

| Name             | Type     | Required |
| :--------------- | :------- | :------: |
| **initialValue** | `number` |    ✔     |
| **timePassed**   | `number` |    ✔     |
| **halfLife**     | `number` |    ✔     |
