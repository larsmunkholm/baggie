Calculate the difference between two timestamps.

## Parameters

| Parameter    | Type     | Required | Description                  |
| :----------- | :------- | :------: | :--------------------------- |
| **fromTime** | `number` |    ✔     | A timestamp in milliseconds. |
| **toTime**   | `number` |    ✔     | A timestamp in milliseconds. |
