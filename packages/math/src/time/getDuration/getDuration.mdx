# getDuration

Get the duration when supplying speed and distance.

If the speed is 100 (meters per minute) and distance is 10 (meters),
then the duration will be 10 (minutes).

## Parameters

| Parameter    | Type     | Required |
| :----------- | :------- | :------: |
| **speed**    | `number` |    ✔     |
| **distance** | `number` |    ✔     |
