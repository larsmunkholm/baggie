# floorDecimals

Round down a number to the nearest decimal places.

## Parameters

| Parameter            | Type     | Required |
| :------------------- | :------- | :------: |
| **number**           | `number` |    ✔     |
| **maxDecimalPlaces** | `number` |    ✔     |

## Examples

Round down to **two** decimals.

<Example number={13.345} maxDecimalPlaces={2} />

Round down to **three** decimals.

<Example number={13.1111} maxDecimalPlaces={3} />

With **negative** decimal places.

<Example number={139.5} maxDecimalPlaces={-2} />
