# roundDecimals

Round a number to the nearest decimal places.

## Parameters

| Parameter            | Type     | Required |
| :------------------- | :------- | :------: |
| **number**           | `number` |    ✔     |
| **maxDecimalPlaces** | `number` |    ✔     |

## Examples

Round to **two** decimals.

<Example number={13.3689} maxDecimalPlaces={2} />

Round to **three** decimals.

<Example number={-13.1124} maxDecimalPlaces={3} />

With **negative** decimal places.

<Example number={139.5} maxDecimalPlaces={-2} />
