# clamp

Make sure a number is within the range of a minimum and maximum value.

If the number is smaller or larger, the nearest accepted value will be
returned.

## Parameters

| Parameter  | Type                           | Required | Descriptions                                                                    |
| :--------- | :----------------------------- | :------: | :------------------------------------------------------------------------------ |
| **input**  | `number`                       |    ✔     |                                                                                 |
| **minMax** | `{ min: number; max: number }` |    ✔     | Both `min` and `max` are optional, but at least one of them should have a value |
