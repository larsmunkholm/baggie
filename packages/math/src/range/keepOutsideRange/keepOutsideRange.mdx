# keepOutsideRange

Make sure a number is outside the range of a minimum and maximum value.

If the number is inside the range, the nearest accepted value will be
returned.

## Parameters

| Parameter         | Type                                                                               | Required | Default value |
| :---------------- | :--------------------------------------------------------------------------------- | :------: | :------------ |
| **number**        | `number`                                                                           |    ✔     |               |
| **ranges**        | An `Array` or single value of `{ from: number; to: number }` or `[number, number]` |    ✔     |               |
| **combineEquals** | `boolean`                                                                          |          | `true`        |
