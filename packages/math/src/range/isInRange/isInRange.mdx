# isInRange

Check if a number is within the range of a minimum and maximum value.

The function accepts multiple ranges to check the number against, and
returns `true` if the number is within the bounds of at least one of
the ranges.

## Parameters

| Parameter  | Type                                                 | Required |
| :--------- | :--------------------------------------------------- | :------: |
| **number** | `number`                                             |    ✔     |
| **ranges** | An `Array` or single value of `{ from: number; to: number }` or `[number, number]` |    ✔     |
