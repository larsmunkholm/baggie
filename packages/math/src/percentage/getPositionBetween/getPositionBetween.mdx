# getPositionBetween

Get the position between two numbers from a percentage.

## Parameters

| Parameter      | Type     | Required |
| :------------- | :------- | :------: |
| **from**       | `number` |    ✔     |
| **to**         | `number` |    ✔     |
| **percentage** | `number` |    ✔     |
