# getAnalogousColorScheme

Get an analogous color scheme.

These are colors that are next to each other on the color wheel.

## Parameters

| Parameter | Type     | Required | Default value | Description                                                                                                    |
| :-------- | :------- | :------: | :------------ | -------------------------------------------------------------------------------------------------------------- |
| **color** | `string` |    âœ…    |               | The base color.                                                                                                |
| **count** | `number` |          | `3`           | The amount of colors you want (including the base color).                                                      |
| **steps** | `number` |          | `12`          | The number of steps you want the color spectrum split into.<br />**Should be equal to or larger than `count`** |

Returns an array of hex colors.
