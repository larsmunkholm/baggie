# normalizeHexColor

Make sure a hex color is in lowercase with 6 to 8 hexadecimals.

A use case for this could be the HTML color input `<input type="color" />` which
requires its value to be in the format `#rrggbb` where rr, gg, bb are two-digit
hexadecimal numbers.
