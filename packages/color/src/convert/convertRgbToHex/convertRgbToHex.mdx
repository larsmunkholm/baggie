# convertRgbToHex

Convert an object with RGB color values to a hex color.

## Parameters

| Parameter | Type     | Required |
| :-------- | :------- | :------: |
| **color** | `object` |    ✔     |

| Color property | Type     | Required | Description                   |
| :------------- | :------- | :------: | :---------------------------- |
| **red**        | `number` |    ✔     | The red color value (0-255)   |
| **green**      | `number` |    ✔     | The green color value (0-255) |
| **blue**       | `number` |    ✔     | The blue color value (0-255)  |
| **alpha**      | `number` |          | The alpha value (0-1)         |
