# baggie

The little bag-of-tricks.

## Technologies

- ✅ All code is in the **[TypeScript](https://www.typescriptlang.org/)** language when possible
- 📚 We use **[Storybook 7](https://storybook.js.org/)** for testing and documentation
- 👷 Our build tool is **[Vite](https://vitejs.dev/)** (using Rollup and SWC)
- 🧹 Everything is linted with **[ESLint](https://eslint.org/)** and **[Stylelint](https://stylelint.io/)**
- 📐 For consistent code formatting, we use **[Prettier](https://prettier.io/)**
- 🪝 Git hooks are set up using **[husky](https://typicode.github.io/husky/)** and **[lint-staged](https://github.com/okonet/lint-staged)**
